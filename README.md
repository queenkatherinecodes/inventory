1. Open Command Prompt or PowerShell
2. Navigate to the project directory:
   ```
   cd C:\path\to\inventory
   ```

3. Create a virtual environment:
   ```
   python -m venv venv
   ```

4. Activate the virtual environment:
   ```
   .\venv\Scripts\activate
   ```

5. Install dependencies:
   ```
   pip install pytest
   ```

6. Run the tests:
   ```
   pytest -v
   ```
